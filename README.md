Save the following variables in a `.env` file in `./EVM_Tracker`:
```
TOKEN=<telegram-token>
CHAT_ID_ALERTS=<chat-id>
CHAT_ID_DEBUG=<chat-id>

WEB3_INFURA_ETHEREUM=https://mainnet.infura.io/v3/<your-key>
WEB3_INFURA_OPTIMISM=https://optimism-mainnet.infura.io/v3/<your-key>
WEB3_INFURA_ARBITRUM=https://arbitrum-mainnet.infura.io/v3/<your-key>
WEB3_INFURA_POLYGON=https://polygon-mainnet.infura.io/v3/<your-key>

ETHEREUM_API_KEY=<your-key>
OPTIMISM_API_KEY=<your-key>
ARBITRUM_API_KEY=<your-key>
POLYGON_API_KEY=<your-key>
GNOSIS_API_KEY=<your-key>
```
